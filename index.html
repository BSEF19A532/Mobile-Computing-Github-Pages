How to Upload static pages to 'Github Pages'

The first thing you need to do is create a Github repository. After creating a Github repository, we need to push an index.html file in the root folder. We can push static content along with it as well.

After making sure, that our repository contains the desired static files. We need to go to the 'Settings' tab on the our Github repository page. After that we need to select the Pages tab from the left column. Selecting the Pages tab will display the appropriate content on the right side.

Now we need to specifiy the name of the branch that contains our static files. After selecting the branch, click save. Once done, a link will appear beside the content linking to the static files that were pushed in the repository.
